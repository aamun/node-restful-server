# Resful Api using Node.js with Express and Mongoose 

It is a example of Resful Api for users accounts.

### Requirements
- Node.js
- Npm
- MongoDB
- Mongoose
- Express

### Instructions

1. Clone the repository

2. Install dependencies
	npm install

3. Run MongoDB server

4. Run web server application
	node server.js
